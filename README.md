SOURDI est une plateforme web de motivation pédagogique basée sur la gamification.

Elle permet aux élèves de :

* Gagner des Sourdi Coins grâce à des KPI pédagogiques (leçons complétées, régularité, temps passé sur la plateforme).
* Utiliser ces coins dans une marketplace éducative intégrée pour accéder à des récompenses ou des contenus pédagogiques.

Le projet utilise :

* Une architecture microservices
* MongoDB pour la gestion de la base de données
* Node.js et Express.js pour le backend
* React.js pour le frontend

Projet réalisé dans le cadre d’un Projet de Fin d’Études.
