**SOURDI** est une plateforme web de motivation pédagogique basée sur la gamification.

Elle permet aux élèves de :

* Gagner des **Sourdi Coins** grâce à des KPI pédagogiques (leçons complétées, régularité, temps passé).
* Utiliser ces coins dans une marketplace éducative intégrée.

Le projet utilise :

* Une architecture **microservices**
* **.NET** (backend)
* **YARP** (API Gateway)
* **React** (frontend)

Cette architecture garantit :

* La sécurité
* L’évolutivité
* La maintenabilité

Projet réalisé dans le cadre d’un Projet de Fin d’Études (PFE).
